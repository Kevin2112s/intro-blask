#### Crear entorno virtual
```python
python /m venv .env
```

#### activamos el entorno virtual
```shell
.\env\Scripts\activate
source .env/bin/activate
```

#### Mostrar paquetes instalados
```python
pip freeze
pip freeze > paquete.txt
``` 

#### En caso de recrear el proyecto
```
pip install -r paquetes.txt
```
